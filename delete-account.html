Konto löschen – Signalix

Informationen und Kontaktmöglichkeit zur Löschung eines Signalix-Kontos und zugehöriger Daten.

Stand: 21.03.2026

1. Anfrage zur Kontolöschung

Nutzerinnen und Nutzer von Signalix können die Löschung ihres Kontos sowie die Löschung der zugehörigen personenbezogenen Daten anfragen.

Zur Anforderung einer Kontolöschung sende bitte eine E-Mail an:

Ricardo Pinto da Costa
E-Mail: signalix.kontakt@gmail.com
Betreff-Empfehlung: Signalix – Konto löschen

2. Welche Angaben in der Anfrage enthalten sein sollten

Damit die Anfrage korrekt zugeordnet werden kann, sollte die E-Mail möglichst Folgendes enthalten:

  • die bei Signalix verwendete Telefonnummer
  • optional weitere Angaben zur Identifikation, falls erforderlich
  • eine klare Erklärung, dass das Signalix-Konto gelöscht werden soll
Wichtiger Hinweis: Zur Vermeidung unbefugter Löschungen kann vor der endgültigen Bearbeitung eine zusätzliche Verifizierung der Identität erforderlich sein.

3. Folgen der Kontolöschung

Nach erfolgreicher Bearbeitung der Anfrage wird das betroffene Signalix-Konto deaktiviert bzw. gelöscht. Soweit anwendbar, werden zugehörige personenbezogene Daten im Rahmen gesetzlicher Vorgaben gelöscht oder gesperrt.

Bestimmte Daten können gegebenenfalls für einen begrenzten Zeitraum weiter gespeichert werden, sofern dies aus rechtlichen, sicherheitsbezogenen oder technisch notwendigen Gründen erforderlich ist.

4. Bearbeitungszeit

Anfragen zur Kontolöschung werden innerhalb eines angemessenen Zeitraums bearbeitet. Falls zusätzliche Informationen zur Verifizierung erforderlich sind, kann sich die Bearbeitungszeit entsprechend verlängern.

5. Weitere Informationen

Weitere Informationen zur Verarbeitung personenbezogener Daten findest du in der Datenschutzerklärung von Signalix.
